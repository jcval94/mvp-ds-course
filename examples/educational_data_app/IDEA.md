# IDEA: Educational Data App

## Nombre provisional

Histohistorias

## Idea en una frase

Una app educativa autocontenida para enseñar histogramas con storytelling y datos pequenos de ejemplo.

## Problema que resuelve

Estudiantes confunden histogramas con graficas de barras y no entienden como interpretar distribuciones.

## Usuario objetivo

Docentes introductorios de ciencia de datos y estudiantes principiantes.

## Resultado esperado

El usuario entiende que muestra un histograma, cambia bins y explica la forma de una distribucion.

## Contexto

Debe funcionar sin cuentas y con ejemplos incluidos.

## Plataforma deseada

Web app simple.

## Restricciones

- Sin login.
- Sin LMS.
- Sin datos externos.
- Sin seguimiento de estudiantes.

## Lo que NO debe incluir el MVP

- Curso completo.
- Multiusuario.
- IA generativa.
- Panel administrativo.

## Supuestos permitidos

El agente puede asumir estudiantes de nivel inicial y un dataset pequeno incluido.
