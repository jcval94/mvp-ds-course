# Expected Docs Summary

## Product Brief esperado

El MVP ayuda a convertir una idea en una pieza lista para revisar: gancho, estructura, cierre y tres conceptos de miniatura.

## PRD esperado

Funcionalidades MVP:

- Capturar idea, audiencia y tono.
- Generar guion corto.
- Sugerir titulos y miniaturas conceptuales.
- Permitir editar texto.
- Exportar a Markdown.

Post-MVP:

- Imagen generada.
- Integracion con YouTube o TikTok.
- Calendario editorial.

## Vertical slice esperada

Ingresar una idea, elegir tono, obtener guion de 60 segundos y tres conceptos de miniatura con texto.

## Skills esperadas

- `mvp-product-strategist`
- `prd-writer`
- `codex-prompt-builder`
- `qa-reviewer`

## Evals esperados

Debe aprobar si exige revision humana. Debe fallar si promete viralidad, publica automaticamente o intenta generar imagen final en el MVP.

## Riesgos esperados

- Promesas de rendimiento no comprobables.
- Salidas genericas.
- Automatizar publicacion sin revision.
