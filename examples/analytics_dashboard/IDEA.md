# IDEA: Analytics Dashboard

## Nombre provisional

LeadRank Lite

## Idea en una frase

Un dashboard interno para priorizar leads comerciales usando CSV y reglas simples.

## Problema que resuelve

El equipo comercial pierde tiempo revisando leads sin saber cuales tienen mayor probabilidad de conversion.

## Usuario objetivo

Coordinador comercial de un equipo B2B pequeno.

## Resultado esperado

Ver un top de leads priorizados con razones claras para contactar primero.

## Contexto

Los datos iniciales vienen de exportaciones CSV.

## Plataforma deseada

Dashboard web local.

## Restricciones

- Sin CRM en MVP.
- Sin modelo predictivo entrenado.
- Sin permisos complejos.
- Sin escritura de vuelta a sistemas externos.

## Lo que NO debe incluir el MVP

- Salesforce.
- Automatizacion de correos.
- Forecast avanzado.
- Gestion completa de pipeline.

## Supuestos permitidos

El agente puede definir scoring por reglas simples y datos de ejemplo.
