# Examples

## Ejemplo canónico

`educational_data_app/` representa el tipo de solicitud que DataClass Forge debe procesar: material educativo de ciencia de datos con visualización, práctica y alcance reducido.

## Fixtures históricos

`ai_content_tool/` y `analytics_dashboard/` provienen de la etapa en que el repositorio era una fábrica genérica de MVPs. Se conservan únicamente para detectar regresiones de migración y no definen el producto actual.

Los nuevos ejemplos deben estar relacionados con:

- conceptos de ciencia de datos;
- diseño curricular;
- módulos visuales;
- ejercicios dependientes de evidencia;
- paquetes para enseñar en vivo.
