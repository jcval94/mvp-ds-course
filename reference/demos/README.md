# Demos históricos de referencia

Esta carpeta es histórica y no bloqueante. El MVP actual de DataClass Forge no depende de demos externos.

Si algún día se agregan demos, pueden usarse solo como inspiración visual:

- `concepro_histograma.html`
- `ejercicios_histograma.html`

El MVP generado en `generated/data-class-problem-generator/` ya incorporó patrones como:

- Visualizaciones animadas.
- Explicacion paso a paso.
- Barras que crecen.
- Datos que aparecen visualmente.
- Comparacion de patrones.
- Interactividad por clic.
- Estetica dark dashboard.
- Casos narrativos con rol, pregunta, feedback, pistas y conclusion.
- Estructura tipo `APP_CONFIG`.
- App autocontenida en un unico HTML.
