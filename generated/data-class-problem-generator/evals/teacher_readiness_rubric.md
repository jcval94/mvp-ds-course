# Teacher Readiness Rubric

| Dimensión | 1 | 3 | 5 |
| --- | --- | --- | --- |
| Guion | No guía la clase | Secuencia básica | Minuto a minuto claro |
| Prompt Codex | Vago | Útil pero incompleto | Ejecutable, técnico y con criterios |
| Prompt Gemini | Genérico | Explica algo | Incluye pasos, preguntas, errores y plan B |
| Notebook blueprint | Ausente | Estructura mínima | Listo para convertir a notebook |
| Dataset | Insuficiente | Datos básicos | Columnas, filas, lógica y ejemplos |
| Código Python | Ausente | Código parcial | Demo local, simple y sin datos externos |
| Plan offline | Ausente | Menciona alternativa | Usable si falla IA o demo |
| Cierre pedagógico | Débil | Resumen simple | Insight, errores y evaluación |
| Checklist docente | Ausente | Lista parcial | Preparación clara antes de enseñar |

## Criterio de aprobación

Promedio 4 o mayor y sin bloqueos críticos.
