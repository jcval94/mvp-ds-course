# Learning Module Quality Checklist

- [ ] El módulo tiene objetivo claro.
- [ ] Explica la intuición del concepto.
- [ ] Tiene visualización obligatoria.
- [ ] La visualización representa el concepto.
- [ ] Incluye secciones breves y progresivas.
- [ ] Incluye error común.
- [ ] Incluye mini-checkpoint.
- [ ] El feedback del checkpoint enseña.
- [ ] Incluye transición a práctica.
- [ ] No es texto largo sin interacción.

## Fallar si

- No hay visual.
- La explicación no menciona la intuición.
- El checkpoint se puede responder sin mirar el concepto.
