# Visual Concept Rubric

Evalúa cada concepto de 1 a 5.

| Dimensión | 1 | 3 | 5 |
| --- | --- | --- | --- |
| Representación visual | No existe | Existe pero es débil | Representa claramente el concepto |
| Elementos mínimos | Faltan elementos clave | Incluye algunos | Incluye todos los requeridos |
| Interacción | No aporta | Decorativa | Desbloquea evidencia o insight |
| Simplificación | Distorsiona | Simplifica con dudas | Simplifica sin romper aprendizaje |
| Tres modos | Solo un modo | Dos modos | Aprender, Ejercitar y Enseñar en vivo |

## Bloqueos

- Concepto sin `visualSpec`.
- Visual que contradice explicación.
- Visual que no puede renderizarse.
- Concepto que no genera alguno de los tres modos.
