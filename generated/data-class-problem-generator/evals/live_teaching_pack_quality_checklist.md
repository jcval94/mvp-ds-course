# Live Teaching Pack Quality Checklist

- [ ] Tiene resumen de clase.
- [ ] Organiza artefactos en tabs o acordeón.
- [ ] Tiene guion minuto a minuto.
- [ ] Recomienda visualización a construir.
- [ ] Incluye dataset sintético o estructura de datos.
- [ ] Incluye prompt para Codex.
- [ ] Incluye prompt para Gemini.
- [ ] Incluye notebook blueprint.
- [ ] Incluye prompt para demo HTML.
- [ ] Incluye prompt para ejercicios adicionales.
- [ ] Incluye assessment prompt.
- [ ] Incluye plan offline.
- [ ] Incluye checklist de preparación del profesor.
- [ ] Incluye checklist durante clase.
- [ ] Incluye errores comunes del alumno.
- [ ] Incluye cierre pedagógico.
- [ ] Incluye código Python sugerido.
- [ ] Dataset muestra columnas, tipos, primeras filas y lógica.
- [ ] Permite copiar prompts.
- [ ] Permite copiar cada sección relevante.
- [ ] Permite descargar paquete JSON.
- [ ] El prompt para Codex tiene restricciones técnicas y criterios de aceptación.
- [ ] El prompt para Gemini tiene preguntas socráticas y posibles errores.

## Fallar si

- Intenta ejecutar IA dentro del navegador.
- El prompt para Codex no tiene criterios de aceptación.
- El prompt para Gemini no anticipa errores de estudiantes.
- El código Python depende de datasets externos.
- El notebook blueprint no tiene celdas ordenadas.
