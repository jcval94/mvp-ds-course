## ConceptSpec

- **Nivel/bloque:** 6, Error de regresión.
- **Objetivo:** Comparar contra predecir el promedio.
- **Definición:** R² compara error del modelo con error de una línea base promedio.
- **Intuición:** El visual hace visible reducción relativa de error frente a baseline antes de resumirlo.
- **Error plausible:** Convertir una salida exploratoria o una métrica aislada en una decisión automática.
- **Mecanismo visual:** reducción relativa de error frente a baseline.
- **Estados:** Antes de decidir → Evidencia revisada.
- **Unidad:** una observación es una noche del puesto.
- **Variables:** pedidos_totales, prediccion_pedidos.
- **Límite:** El test se usa una sola vez; las métricas describen este procedimiento y no prueban causalidad.

## LearningModule

1. Paco pone el promedio como rival.
2. Ejecutar **Comparar estados de r²**.
3. Cita una marca visible, responde la pregunta de Don Juan y nombra un límite.

## PracticeExercise

- **Guiado:** Observa modelo frente a promedio base y cita una marca visible. ¿Qué conclusión guiada sobre r² conserva el alcance?
- **Transferencia:** Observa modelo frente a promedio base y cita una marca visible. ¿Qué exige transferir r² a otra decisión?
- **Bloqueo:** 1 cambios y todas las marcas requeridas.
- **Separación:** Aprender revela el mecanismo; los dos ejercicios usan incidentes y evidencia nuevos.
