## ConceptSpec

- **Nivel/bloque:** 7, Métricas de clasificación.
- **Objetivo:** Combinar precision y recall armónicamente.
- **Definición:** F1 es la media armónica de precision y recall.
- **Intuición:** El visual hace visible equilibrio que penaliza un valor muy bajo antes de resumirlo.
- **Error plausible:** Convertir una salida exploratoria o una métrica aislada en una decisión automática.
- **Mecanismo visual:** equilibrio que penaliza un valor muy bajo.
- **Estados:** Antes de decidir → Evidencia revisada.
- **Unidad:** una observación es una noche del puesto.
- **Variables:** precision, recall.
- **Límite:** El test se usa una sola vez; las métricas describen este procedimiento y no prueban causalidad.

## LearningModule

1. Paco compara dos reglas con compromisos distintos.
2. Ejecutar **Comparar estados de f1**.
3. Cita una marca visible, responde la pregunta de Don Juan y nombra un límite.

## PracticeExercise

- **Guiado:** Observa precision y recall combinados y cita una marca visible. ¿Qué conclusión guiada sobre f1 conserva el alcance?
- **Transferencia:** Observa precision y recall combinados y cita una marca visible. ¿Qué exige transferir f1 a otra decisión?
- **Bloqueo:** 1 cambios y todas las marcas requeridas.
- **Separación:** Aprender revela el mecanismo; los dos ejercicios usan incidentes y evidencia nuevos.
